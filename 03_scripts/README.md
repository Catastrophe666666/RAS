# Scripts

This folder contains scripts needed to reproduce or extend the current workflow.

- `data_processing/`: extraction, reconciliation, text dataset construction, and topic-modeling dataset construction.
- `topic_modeling/`: TF-IDF, BERTopic, and STM scripts.
- `network_analysis/`: author-subject network and subject evolution scripts.

Scripts associated only with incomplete or excluded branches, such as CorEx and cross-model comparison templates, are not included.
