# Metadata

This folder contains the core metadata files used to identify RAS articles and connect them to subject categories.

- `1858_1948_full_subject.csv`: broad article-subject metadata.
- `1858-1948_author_subject.csv`: bried article-subject metadata
- `merge_index_and_master.csv`: combined index and master metadata.
- `ras_subject_expanded.csv`: current main metadata file used by text and network workflows, produced by rmd.
